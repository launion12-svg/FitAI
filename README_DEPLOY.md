# FitAI – Deploy
1) Sube estos archivos a la raíz del repo en GitHub.
2) En Vercel → Settings → Environment Variables añade: GEMINI_API_KEY=tu_clave.
3) Redeploy con *Clear build cache*.
4) Prueba /api/gemini?ping=1 → debe devolver ok:true.
5) Abre la web: verás la pantalla de Bienvenida → Formulario → Dashboard (según tu HTML original).
